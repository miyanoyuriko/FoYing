Decade 我以为我会尖酸-驰名荼菜汤


<p dir="ltr"  >Decade</p> 
<p dir="ltr"  >我以为我会尖酸刻薄地嘲讽他十年如一日的所谓绅士风度，谁知预想中的千言万语，最后出口的却只是一句，<br /> &nbsp; “是你啊。”</p> 
<p dir="ltr"  >&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ——序</p> 
<p dir="ltr"  >&nbsp; 我在表格的姓名栏上，郑重地填上我的名字:“弗朗西斯.波诺弗瓦”。<br /> &nbsp; 坐在对面的伊丽莎白.海德薇莉女士理了理棕色卷发，“好的，波诺弗瓦先生，那么下面请开始说您的故事吧。不过老实说，我可真想不到，您居然会有一段令您难以忘怀的恋情。毕竟我看不出您这么深情。”<br /> &nbsp; 我被她逗笑了，我给她的形象莫过于风流成性的情场浪子，挚爱一人永藏于心根本就不符合她对我的定义。<br /> &nbsp; “哥哥我怎么会深情。”我浅啜一口拿铁，感受口腔回荡的苦涩，“我只是忘不了他。”<br /> &nbsp; 他，亚瑟.柯克兰，难以忘怀的他。<br /> &nbsp; 我甚至都不知道我们是怎么成为恋人的，用安东尼奥的话来说，我们的关系比烂熟的番茄还要烂，烂到已经腐蚀入骨，血肉相融。<br /> &nbsp; 还是孩子的时候，我们就在互掐打架，他穿着格子短裤，纤细的小腿在我眼前一晃，把我踹下池塘，我不甘示弱向他泼水，惹得从小以绅士自居的他抽出雨伞把我的头发卷在伞柄上，蹬着绿眼睛嘲笑我。<br /> &nbsp; 他的眼睛和脾性像狼，但是遇到我会变成哈士奇。<br /> &nbsp; 他在我的日记本上看到这句话，把我引以为傲的脸蛋揍肿了。<br /> &nbsp; 我的朋友，亚瑟.柯克兰，死板生硬不解风情，口是心非毒舌暴力，厨艺烂透性格别扭……他的缺点我可以罗列一张表。<br /> &nbsp; 在我十五岁的时候，我第一次意识到我对亚瑟有某种不同于朋友的情愫。<br /> &nbsp; 那是个平常的下午，我坐在地上打游戏，他躺在沙发上舔冰淇淋，电扇转动着扇叶吹出丝丝凉风，吹起他短袖上衣的一角，酷热的天气让冰淇淋的边角融化，空气中挥发着甜腻的香气，他鲜红的舌尖上蘸着汁液，唇角边残留着细碎的巧克力。<br /> &nbsp; 我突然很想亲上去。<br /> &nbsp; 我被这念头吓了一跳，啪地把手柄拍在地上，他翻了个白眼，“发什么疯!”<br /> &nbsp; 我迅速回嘴，“我的绅士，请你擦擦嘴，还有不要把冰淇淋滴在沙发上，我懒得拆开洗。”<br /> &nbsp; 他恼羞成怒地用手背往嘴上一擦，转过后“哼”一声。<br /> &nbsp; 我盯着他发红的脸颊和耳垂，继而心慌意乱地转移注意力。<br /> &nbsp; 我很讨女孩子的喜欢，从十二岁开始我就收到情书，当然这归功于我的人格魅力（容许我自恋一会儿）。亚瑟扯着眼角，“我真应该劝她们去配眼镜，给一只提早发情的孔雀写情书。”<br /> &nbsp; 我像牛皮糖一样黏在他身上，借此嗅他身上若有若无的气味，“噢亚瑟，我可羡慕你。”<br /> &nbsp; “什么?”<br /> &nbsp; “你有一个这么英俊潇洒的朋友。”<br /> &nbsp; “滚。”他没好气地推开我，结果阴差阳错撞入我的怀里。<br /> &nbsp; 他的头发扫过我的下颚，后背挨着我的胸膛。<br /> &nbsp; 这一瞬间，我开始贪婪，妄想我们的关系不应该只限友情。<br /> &nbsp; 我擒住他的手臂，在他迷茫的眼神里，含住了他的唇瓣。<br /> &nbsp; 他愣了愣，但没有反抗。<br /> &nbsp; 我们都很青涩，我甚至撞到了他的牙齿，他也咬住了我的舌头。分开的时候我们狼狈地捂着嘴，相视着噗嗤一笑。<br /> &nbsp; 他笑得眼角飚泪，“我的天你不是吹嘘自己是情圣吗!你的吻技也太烂了!”<br /> &nbsp; “你也好不到哪儿去，哥哥我的舌头都要贴创可贴了。”<br /> &nbsp; 我们在一起仿佛是顺理成章的事情，好似上天为我们的人生安排了一个“青梅竹马相恋”的程序，然后在这一天，我们的身体接受到指令，开始执行。<br /> &nbsp; 恋爱的日子我至今仍视为至宝，我们躲在厨房的橱柜后偷偷接吻，他的舌尖故意点在我的伤口上，带着恶趣味的笑容看我疼得呼气，发出“嘶嘶”的声音。<br /> &nbsp; 我叫他小眉毛小亚蒂小毛虫小恶魔小宝贝，他做出呕吐状嘴角抽搐说我把他恶心到了，我抱着他在他的耳边不停地叫这些可爱的绰号，他翻着白眼却掩饰不住笑意。<br /> &nbsp; 课后无人的教室里，花园新剪的草坪上，面包店的货架旁，我们挥洒着我们年少的青春。<br /> &nbsp; 我的上帝，这是我这一生的最甜蜜的时光。</p> 
<p dir="ltr"  >&nbsp; 程序总有坏掉亦或是不再启动的一天。<br /> &nbsp; 亚瑟不是个好脾气的情人，虽然我很喜欢他吃醋的模样。<br /> &nbsp; 他开始学会冷淡和漠视来抗议向我冲来的狂蜂浪蝶，我宁愿他讽刺我甚至揍我，他没有。<br /> &nbsp; 他只说，“你太滥情了。”<br /> &nbsp; 这一年我们已然成年，我也不再期盼他像冲动的孩子一样，可怕的成长让我圆滑，一旦我疏远那些花朵般娇艳的姑娘们，她们脸上的泪水和哀怨总让我于心不忍。<br /> &nbsp; 我说我依旧爱你，他淡淡点头，翻着手上的文件。<br /> &nbsp; “我把太多的岁月耗在你身上了，这不值得，弗朗西斯。”他看着远方向我飞吻的女孩，轻描淡写地说。<br /> &nbsp; 我粗暴地把他压在墙上亲吻，血从他的口中渗出，我疯狂地汲取他的血液和他，我开始恐慌，我有预感，我即将失去他。<br /> &nbsp; 果不其然，九月，他回到家乡伦敦，宣告他大学生活的开始和我们恋情的结束。<br /> &nbsp; “弗朗西斯，很高兴我们有三年的愉快时光，谢谢你。但我想我们也许只限友情，过多的接触会产生可怕的控制欲，我已经病入膏肓，再见。”<br /> &nbsp; 客气的用词让我怀疑他是从网上抄的。<br /> &nbsp; 我每天都偷偷地看他的ins账号，看他琐碎的日常——他加入了乐队，还加入了好笑的魔法部，有了新的校友和新的生活。<br /> &nbsp; 这一切都没有弗朗西斯.波诺弗瓦。<br /> &nbsp; 我想起我们还小的时候，我挥舞着他的魔法棒一路奔跑，我说我要把自己变成会飞的精灵，跟在他的身边捉弄他。<br /> &nbsp; 我翻箱倒柜，翻出破旧的魔法棒，不停地在自己的头上转动，转着转着翅膀没长出来，眼泪却先滴落在地。<br /> &nbsp; 一晃十年。<br /> &nbsp; 机缘巧合，一位叫王耀的东方客户跟我谈起亚瑟，他说他是亚瑟的校友，他说亚瑟经常提起我。<br /> &nbsp; 我找了间清净的小酒馆约王耀出来，开门见山，“亚瑟和你说了什么?他……现在怎么样?”<br /> &nbsp; 他勾着唇角笑得神秘，“你怎么知道他会和我说什么?”<br /> &nbsp; “他不会和不信任的人聊任何关于他自己的事情。”<br /> &nbsp; “你可真了解他。”王耀笑了笑。“他中毒了，病毒叫弗朗西斯.波诺弗瓦。”<br /> &nbsp; “知道我怎么认识亚瑟的吗？他来的第一天，就喝醉了酒，不停地嚷着弗朗西斯，也就是你的名字。我把他扶回宿舍，照看着他。”<br /> &nbsp; 我沉默地喝着酒。<br /> &nbsp; “详细的相处我想你没兴趣听，他说，他无法忍受你把目光转移到别人身上，即使他知道你爱他；他说自己简直就是个变态，想任性地占有你，太过强烈的爱意无法让他理智。他选择了自救。”<br /> &nbsp; 原来我爱他是毁了他，我是他难以根除的病毒。<br /> &nbsp; 我颓然地坐在椅子上，看着王耀走出酒馆，看着日落而息月华初上。我不知道呆了多久，直到服务生告诉我酒馆要打烊了。<br /> &nbsp; 我像个乞丐一样在街上飘荡。我何尝不是呢?我是爱情的乞丐，我一无所有。<br /> &nbsp; 上天又给我开了一个玩笑，此时此刻，整整十年后，我在巴黎街头遇到了我曾经的恋人亚瑟.柯克兰。<br /> &nbsp; 我想我一定十分狼狈，以至于他的眼睛里出现了错愕。<br /> &nbsp; 我无数次幻想过这个场景，现在他真实地出现在我的眼前。<br /> &nbsp; 我们擦肩而过，我收回了向他伸出的手。</p> 
<p dir="ltr"  >&nbsp; “这就是我珍藏的故事。”<br /> &nbsp; 我叹了口气，拒绝了伊丽莎白递过来的纸巾。<br /> &nbsp; 她收拾表格向我道别，推开门踏出咖啡馆外时，我突然站起。<br /> &nbsp; “请帮我加一句话!”<br /> &nbsp; “什么?”<br /> &nbsp; “我的爱人有一对熠熠生辉的祖母绿色眼睛。”我哽咽着。&nbsp; <br /> &nbsp; “闪耀了我的一生。”</p>
